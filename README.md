# codingtest-study

백준 알고리즘 문제를 풀며 유형별로 정리한 저장소입니다.  
단순 정답 저장이 아니라, 문제 해결 과정과 실수 포인트까지 함께 기록합니다.

---

## 📌 목표
- 코딩테스트 대비
- 자료구조 & 알고리즘 기본기 강화
- 문제 해결 과정 기록 습관 만들기
- 자주 틀리는 유형 복습

---

## 🧩 진행 방식
- 문제 풀이 후 유형별로 분류
- 코드 상단에 핵심 아이디어 작성
- 틀린 문제는 오답노트에 기록
- 다시 풀 문제 따로 관리

---

## 🔥 기록 방식 예시

### 백준 2512 예산
- 유형: 이분 탐색
- 핵심 아이디어: 상한액을 mid로 두고 총합 비교
- 실수: total 초기화 위치 실수
- 배운 점: 이분 탐색은 조건 기준이 명확해야 한다

---

## 📌 규칙
- 파일명: `문제번호_문제이름.py`
- 커밋 메시지: `solve: 문제번호 문제이름`
- 하루 최소 1문제
